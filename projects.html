Commute Predictor Machine Learning Model

This project combines my bicycle commute data from Strava with Calgary weather data to train and implement a machine learning model to predict my commute time based on current weather. A GUI was used to input current weather and display my predicted commute time which is accurate to within 1 minute and 38 seconds. This project also uses web scraping to autofill certain cells in the GUI.

Skills: Python, Pandas, Machine Learning, Tkinter

NBA True Player Efficiency Analysis

Having recently become interested in the sport of cycling, I learned that a common measure of a cyclist's ability is their Functional Threshold Power (FTP), which is often measured in Watts/Kilogram. My partner, Sean, and I decided to apply a similar method of analysis on publicly available NBA player tracking data to determine which players are capable of scoring the most points with the least amount of power required to do so.

Skills: Python, Pandas
